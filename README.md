# Anuj Tools — Official Photo Compressor

A browser-only image compression and resizing tool for official website uploads.

## Features

- JPG, PNG, WEBP output
- Target file size in KB
- Optional width and height
- Fit inside or crop to exact size
- Presets for passport/profile/common upload sizes
- Runs fully in the browser

## Deploy on GitHub Pages

Create a new repo, for example:

```bash
photo-compressor
```

Add these files:

- `index.html`
- `style.css`
- `script.js`

Then push:

```bash
git add .
git commit -m "Add official photo compressor tool"
git push
```

Enable GitHub Pages from repository settings.
